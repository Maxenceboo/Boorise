# Description du projet

Boorise est un **ERP nouvelle génération en mode SaaS**.
Il centralise la gestion des processus clés : facturation, devis, CRM, stocks, trésorerie.

## Objectifs
- Simplifier la gestion quotidienne.
- Offrir une solution rapide à déployer et facile à utiliser.
- Devenir une alternative crédible aux ERP lourds pour PME/TPE.

## MVP (Minimum Viable Product)
1. Facturation & devis.
2. CRM simple.
3. Gestion des produits & stocks.

## Vision long terme
Un hub complet intégrant :
- RH et plannings.
- Comptabilité et reporting.
- Prévisionnel de trésorerie.
- Intégrations e-commerce.
- API ouverte pour partenaires.
